# adonis-skills Web

Web UI for the skills catalog, built with Next.js. It provides skill browsing, detail pages, and one-click install command copy.

This app is part of the [adonis-skills](../../) monorepo.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript 5
- **UI Library**: React 19
- **Styling**: Tailwind CSS 4
- **Fonts**: Space Grotesk + IBM Plex Mono

## Local Development

Run from the **monorepo root** (recommended, dependency tasks managed by Turborepo):

```bash
pnpm dev
```

Or run directly in this directory:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
# Run from monorepo root
pnpm build
```

> The `build` task depends on `skills:validate` and `skills:index` first. Turborepo handles the execution order automatically.

## Directory Structure

```txt
apps/web/
├── src/
│   ├── app/
│   │   ├── page.tsx              # Home: skills list
│   │   ├── layout.tsx            # Root layout (metadata, fonts)
│   │   ├── globals.css           # Global styles (CSS variables)
│   │   ├── not-found.tsx         # 404 page
│   │   └── skills/[slug]/
│   │       └── page.tsx          # Skill detail page (dynamic route)
│   ├── components/
│   │   ├── skill-card.tsx        # Skill card component
│   │   └── copy-install-command.tsx  # Copy install command button
│   ├── lib/
│   │   └── skills.ts             # Skill data loading utilities
│   └── generated/
│       └── skills-index.json     # Generated by script, do not edit manually
└── public/                       # Static assets
```

## Skills Index

`src/generated/skills-index.json` is generated by a root-level script. Do not edit it manually.

After updating skills, run this from the monorepo root:

```bash
pnpm skills:index
```
