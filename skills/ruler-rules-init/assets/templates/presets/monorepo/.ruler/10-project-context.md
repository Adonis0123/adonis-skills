## Project Context

- Stack: Turborepo monorepo + TypeScript + pnpm workspaces.
- Package manager: pnpm (workspaces enabled).
- Quality tools: ESLint, TypeScript typecheck, Prettier.

## Key Directories

- `apps/`: deployable application surfaces (web, api, etc.).
- `packages/`: shared libraries and internal packages.
- `scripts/`: automation and repository utilities.
- `turbo.json`: Turborepo pipeline configuration.
